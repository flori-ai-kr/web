# Hazel Admin 전면 리디자인

## 컨셉: "따뜻한 작업실" (Notion/Arc 스타일)

### 컬러
- 배경: 크림 `#FAFAF8`, 카드: 순백 `#FFFFFF` + shadow-sm
- 액센트: Warm Coral, 서브: Sage Green `#8B9D83`
- 텍스트: 차콜 `#2D2D2A`, Muted: `#9B9B93`
- 폰트: Pretendard 단독, Border radius: 12px

### 사이드바
- 3개 섹션: 매장운영(대시보드,캘린더,매출,지출) / 고객기록(고객,사진첩) / 분석(입금대조,통계)
- 활성: 좌측 Coral 바, 접이식 유지

### 대시보드
- 우선순위: 할일 > 월간흐름 > 오늘매출
- 인사말 + 할일카드(미입금,오늘예약,미완료) + 월간차트 + 오늘매출/최근주문

### 캘린더 (신규)
- 월간 캘린더 + 우측 사이드패널
- 예약 → 매출전환 플로우
- 메모 기능
- DB: reservations 테이블

### 리스트 페이지 (매출/지출/고객)
- 날짜별 그룹핑 리스트 (테이블 대신)
- 요약카드 3개 상단
- 클릭 → 슬라이드 패널 (Dialog 대신)
- 카테고리 한글 라벨 (sale_categories 조인)

### 사진첩
- Masonry 레이아웃 (CSS columns)
- 호버 오버레이, 라이트박스

### 통계
- recharts 실제 차트: 도넛, 바, 라인
- 고객 분석 텍스트 카드

### 설정
- 탭 구조: 카드수수료 / 카테고리 / 결제방식 / 태그

### 제거
- next-themes, geist, 다크모드, dark: 클래스

### 구현 순서
1. 디자인 파운데이션 (컬러, 폰트, 다크모드 제거)
2. 사이드바 그룹핑 + 레이아웃
3. 대시보드 재설계
4. 매출/지출/고객 리스트 통일
5. 캘린더 구현
6. 사진첩 + 통계 + 설정
