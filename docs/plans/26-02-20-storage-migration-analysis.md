# 이미지 저장소 마이그레이션 분석 (2026-02-15)

## 요구사항 요약
- 월 1,000장 업로드 × 3MB = ~3GB/월 업로드
- 연간 누적: ~36GB
- 한국 사용자 대상 (빠른 CDN 필요)
- 이미지 최적화 (리사이즈, WebP 변환)
- 현재 Supabase Storage 사용 중

## 대안 비교

### 1. AWS S3 + CloudFront

**장점**
- 글로벌 CDN (한국 엣지 로케이션 다수)
- 무제한 확장성
- 세밀한 권한 제어 (IAM)
- Lambda@Edge로 이미지 최적화 가능

**단점**
- 설정 복잡도 높음 (S3 버킷 정책, CloudFront 배포, IAM 설정)
- 여러 서비스 조합 필요
- 오버엔지니어링 위험

**비용 (월간)**
| 항목 | 계산 | 비용 |
|------|------|------|
| S3 저장 (Standard) | 36GB × $0.023/GB | $0.83/월 |
| PUT 요청 | 1,000장 ÷ 1,000 × $0.005 | $0.005/월 |
| CloudFront 전송 | 10GB × $0.085/GB (첫 10TB) | $0.85/월 |
| GET 요청 (10,000회) | 10,000 × $0.0120/10,000 | $0.012/월 |
| **총 월간 비용** | | **~$1.70/월** |
| **연간 비용** | | **~$20.40** |

**마이그레이션 난이도**: ⭐⭐⭐⭐ (높음)
- AWS SDK 설치, IAM 자격증명 관리
- CloudFront 배포 설정 (약 15-30분 소요)
- 기존 코드 전면 수정 필요

**DX**: ⭐⭐ (복잡)
- 여러 서비스 통합, 권한 관리 복잡

**성능**: ⭐⭐⭐⭐⭐ (최고)
- 글로벌 CDN, 한국 엣지 로케이션
- 캐시 히트율 높음

### 2. Cloudflare R2 + Cloudflare Images

**장점**
- **무료 egress (대역폭 비용 없음)**
- S3 호환 API (마이그레이션 쉬움)
- Cloudflare CDN 자동 통합
- 저렴한 저장 비용
- Cloudflare Images로 자동 최적화 (리사이즈, WebP)

**단점**
- 한국 리전 없음 (가장 가까운: 일본 도쿄)
- Images Transform 별도 비용

**비용 (월간)**
| 항목 | 계산 | 비용 |
|------|------|------|
| R2 저장 | 36GB × $0.015/GB | $0.54/월 |
| Class A 작업 (PUT) | 1,000 ÷ 1,000 × $4.50 | $0.0045/월 |
| Class B 작업 (GET) | 10,000 ÷ 10,000 × $0.36 | $0.0036/월 |
| Egress | | **$0** (무료!) |
| Cloudflare Images (선택) | 1,000장 × $5/100k | $0.05/월 |
| **총 월간 비용** | | **~$0.60/월** |
| **연간 비용** | | **~$7.20** |

**마이그레이션 난이도**: ⭐⭐⭐ (중간)
- S3 호환 API (AWS SDK 사용 가능)
- Cloudflare 계정 + API 토큰 설정
- 기존 코드 약간 수정 (엔드포인트 변경)

**DX**: ⭐⭐⭐⭐ (좋음)
- S3 API 호환, Workers 통합 쉬움

**성능**: ⭐⭐⭐⭐ (우수)
- Cloudflare 글로벌 CDN (한국 서울 POP 있음)
- 무료 대역폭으로 비용 걱정 없음

### 3. Vercel Blob

**장점**
- Vercel 네이티브 통합 (이미 Vercel 사용 중)
- 설정 초간단 (환경변수 자동)
- Edge Network 자동 (빠른 전송)
- Next.js와 완벽 호환

**단점**
- **가장 비쌈**
- 저장 용량 제한 (Hobby: 1GB, Pro: 1TB)
- 벤더 락인 (Vercel 종속)

**비용 (월간)**
| 플랜 | 저장 | 대역폭 | 월 비용 |
|------|------|--------|---------|
| Hobby | 1GB | - | **불가능** (1개월 내 초과) |
| Pro | 100GB 포함 | 1TB 포함 | **$20/월** (기본) |
| Pro 초과 | $0.15/GB | $0.15/GB | **$20 + 추가** |

**1년 운영 시 (Pro 플랜)**
- 기본: $20/월 × 12 = **$240**
- 36GB 저장: 100GB 이내 (추가 비용 없음)
- **연간 총 비용: ~$240**

**마이그레이션 난이도**: ⭐ (매우 쉬움)
- `@vercel/blob` 패키지 설치
- 환경변수 자동 주입
- 코드 수정 최소화

**DX**: ⭐⭐⭐⭐⭐ (최고)
- Next.js 네이티브, 간단한 API

**성능**: ⭐⭐⭐⭐⭐ (최고)
- Vercel Edge Network (한국 포함)
- 자동 최적화

**단점**
- **비싼 가격** (Pro $20/월, 10GB만)
- 스타트업 서비스 (안정성 불확실)
- CDN 커스터마이징 제한

**비용 (월간)**
| 플랜 | 저장 | 대역폭 | 월 비용 |
|------|------|--------|---------|
| Free | 2GB | 제한적 | **무료** (1개월 내 초과) |
| Pro | 10GB | 100GB | **$20/월** |
| 추가 저장 | $2/GB | $0.10/GB | **$20 + $52** |

**1년 운영 시 (Pro + 추가)**
- Pro: $20/월
- 추가 저장: (36GB - 10GB) × $2 = $52/월
- **연간 총 비용: ~$864**

**마이그레이션 난이도**: ⭐⭐ (쉬움)
- `uploadthing` 패키지 설치
- 라우트 핸들러 추가
- UI 컴포넌트 교체 가능

**DX**: ⭐⭐⭐⭐⭐ (최고)
- React 컴포넌트 제공, 간단한 설정

**성능**: ⭐⭐⭐ (보통)
- CDN 제공하지만 상세 불명

### 5. Bunny CDN Storage

**장점**
- **초저가** (가장 저렴)
- 글로벌 CDN 포함
- 이미지 최적화 기능 (Bunny Optimizer)
- 99.9% SLA

**단점**
- 한국 리전 없음 (가장 가까운: 싱가포르, 일본)
- 인지도 낮음 (덜 알려진 서비스)

**비용 (월간)**
| 항목 | 계산 | 비용 |
|------|------|------|
| 저장 (Standard) | 36GB × $0.01/GB | $0.36/월 |
| Asia 대역폭 | 10GB × $0.03/GB | $0.30/월 |
| Optimizer (선택) | 1,000장 × $0.70/10k | $0.07/월 |
| **총 월간 비용** | | **~$0.73/월** |
| **연간 비용** | | **~$8.76** |

**마이그레이션 난이도**: ⭐⭐⭐ (중간)
- REST API 사용 (S3 호환 아님)
- 커스텀 업로드 로직 필요
- FTP/API 키 관리

**DX**: ⭐⭐⭐ (보통)
- API 문서 양호, 대시보드 직관적

**성능**: ⭐⭐⭐⭐ (우수)
- 글로벌 CDN (일본 POP 있음)
- 빠른 전송 속도

- 무료 티어 넉넉 (20GB 저장, 20GB 대역폭)
- CDN 포함

**단점**
- 대역폭 제한 (무료: 20GB/월)
- 변환 요청 제한 (무료: 무제한이지만 rate limit)

**비용 (월간)**
| 플랜 | 저장 | 대역폭 | 변환 | 월 비용 |
|------|------|--------|------|---------|
| Free | 20GB | 20GB | 무제한 | **무료** |
| Paid (필요 시) | $9/월 | 60GB | 무제한 | **$9/월** |

**1년 운영 시**
- 36GB 저장: Free (20GB) 초과 → Paid 필요
- **연간 비용: ~$108** (Paid 플랜)

**마이그레이션 난이도**: ⭐⭐ (쉬움)
- ImageKit SDK 설치
- 업로드 엔드포인트 변경
- URL 변환 로직 추가 (선택)

**DX**: ⭐⭐⭐⭐⭐ (최고)
- SDK 제공, 실시간 변환 편리

**성능**: ⭐⭐⭐⭐⭐ (최고)

## 최종 추천

### 🥇 1순위: Cloudflare R2 + Cloudflare Images

**이유**
- **가격**: 연간 $7.20 (가장 저렴)
- **성능**: Cloudflare 글로벌 CDN (한국 서울 POP 포함)
- **무료 대역폭**: egress 비용 없음 (트래픽 증가해도 안전)
- **이미지 최적화**: Cloudflare Images로 자동 리사이즈, WebP 변환
- **마이그레이션**: S3 호환 API (AWS SDK 재사용 가능)
- **확장성**: 무제한 저장, 무료 대역폭으로 미래 대비

**구현 계획**
1. Cloudflare 계정 생성 + R2 버킷 생성
2. API 토큰 발급 (R2 권한)
3. AWS SDK 엔드포인트만 변경 (`s3.amazonaws.com` → `<account-id>.r2.cloudflarestorage.com`)
4. Cloudflare Images 연동 (선택, URL 변환)
5. 기존 Supabase Storage 데이터 마이그레이션 (일괄 복사)

**예상 작업 시간**: 2-3시간

**추천 상황**: Cloudflare 계정 생성이 어렵거나, REST API 선호 시
