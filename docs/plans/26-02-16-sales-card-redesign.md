# 매출 관리 카드형 UX 개선

## 요약
매출 목록을 테이블에서 카드형으로 전환. 데스크톱 2열, 모바일 1열. 일자별 구분선 추가.

## 카드 레이아웃 (compact, CustomerCard보다 세로 폭 축소)
```
┌─────────────────────────────────────────┐
│ [카테고리배지]  결제배지 · 고객명  ₩금액 │
│ 채널 · 메모...  [📷]              >    │
└─────────────────────────────────────────┘
```
- 1줄: 카테고리 배지 + 결제 배지 + 고객명 + 금액(오른쪽)
- 2줄: 채널 + 메모(truncate) + 사진 아이콘(있을 때만)
- 아바타/Stats 없음 → 2줄로 compact

## 일자별 그룹핑
```
── 2월 15일 (토) · 3건 ₩450,000 ──────
  [카드] [카드]    (데스크톱: 2열)
  [카드]
── 2월 14일 (금) · 2건 ₩300,000 ──────
  [카드] [카드]
```
- 날짜 헤더: 날짜 + 요일 + 건수 + 소계
- 구분선으로 시각적 분리

## 반응형
- 데스크톱(md+): 카드 2열 그리드
- 모바일: 카드 1열
