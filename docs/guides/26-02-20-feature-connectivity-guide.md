## 1. 대시보드

로그인하면 가장 먼저 보이는 화면입니다. 오늘의 현황을 한눈에 파악할 수 있어요.

### 오늘의 요약 카드

| 카드 | 내용 |
|------|------|
| 오늘 매출 | 오늘 등록된 매출 합계 |
| 이번 달 매출 | 이번 달 전체 매출 합계 |
| 이번 달 지출 | 이번 달 전체 지출 합계 |
| 순이익 | 이번 달 매출 - 지출 |

### 입금 미확인 알림

카드 결제 중 아직 입금 확인이 안 된 건이 있으면 알림이 표시됩니다.
클릭하면 입금 대조 페이지로 바로 이동해요.

### 오늘 예약

- 오늘 날짜의 예약 목록이 시간순으로 나열됩니다.
- 각 예약의 시간, 고객명, 예상 금액, 상태(대기/확정/완료/취소)를 볼 수 있어요.
- **대기/확정** 상태인 예약에는 **"매출 등록하기"** 링크가 표시됩니다.
  - 클릭하면 캘린더 페이지로 이동하면서 매출 등록 팝업이 자동으로 열려요.

### 최근 매출

- 가장 최근에 등록된 매출 10건이 표시됩니다.
- 카테고리, 결제방식, 고객명을 빠르게 확인할 수 있어요.

### 월간 분석 (월 선택 가능)

상단의 월 선택기로 원하는 달을 선택하면 해당 월의 분석 데이터를 볼 수 있습니다.

- **고객 통계**: 전체 고객 수, 신규 고객, 재구매 고객
- **카테고리별 매출**: 상품 종류별 매출 비율 (막대 차트)
- **결제방식별 매출**: 카드, 현금, 계좌이체, 네이버페이 비율
- **유입 채널별 매출**: 전화, 카카오톡, 네이버, 도로, 기타
- **지출 카테고리별 현황**: 꽃 매입, 배달, 광고, 임대료 등

## 2. 매출 관리

매출 등록, 조회, 수정, 삭제를 모두 할 수 있는 핵심 페이지입니다.

### 매출 등록하기

1. 매출 페이지 상단의 **[+ 매출 등록]** 버튼 클릭
2. 다음 항목을 입력합니다:

| 항목 | 설명 | 필수 |
|------|------|------|
| 날짜 | 매출 발생 날짜 | O |
| 카테고리 | 상품 종류 (설정에서 관리) | O |
| 금액 | 매출 금액 (콤마 자동 입력) | O |
| 결제방식 | 카드, 현금, 계좌이체, 네이버페이 등 | O |
| 카드사 | 카드 결제 시 카드사 선택 | 카드 결제 시 |
| 예약 채널 | 전화, 카카오톡, 네이버, 도로, 기타 | - |
| 고객명 | 고객 이름 (자동완성 지원) | - |
| 전화번호 | 고객 전화번호 (010-0000-0000 자동 포맷) | - |
| 메모 | 추가 메모 사항 | - |

3. **[등록]** 클릭

> **카드 결제를 선택하면?**
> 카드사를 선택하는 순간, 수수료와 예상 입금액이 자동 계산됩니다.
> 예: 50,000원 카드 결제 (수수료 2.5%) -> 예상 입금액: 48,750원
> 예상 입금일도 영업일 기준으로 자동 계산돼요.

> **고객명을 입력하면?**
> 이름을 입력하기 시작하면 기존 고객 목록이 자동완성으로 나타납니다.
> 기존 고객을 선택하면 전화번호가 자동으로 채워져요.
> 새로운 이름 + 전화번호를 입력하면 고객이 자동으로 등록됩니다.

### 매출 목록 조회

- **월 필터**: 원하는 연/월을 선택해서 해당 월의 매출만 볼 수 있어요.
- **결제방식 필터**: 카드, 현금, 계좌이체, 네이버페이 등으로 걸러볼 수 있어요.
- **카테고리 필터**: 상품 종류별로 걸러볼 수 있어요.
- **검색**: 상품명이나 고객명으로 검색할 수 있어요.

### 매출 요약 카드

목록 상단에 다음 정보가 표시됩니다:

- 전체 매출 합계
- 카드 매출 / 네이버페이 매출 / 계좌이체 매출 / 현금 매출 각각의 합계
- 건수

### 매출 상세 보기

목록에서 매출을 클릭하면 오른쪽에 상세 패널이 열립니다.

- 매출의 모든 정보를 확인할 수 있어요.
- 연결된 고객 정보와 링크
- 카드 수수료 계산 내역
- 입금 상태 (대기/완료/해당없음)
- 연결된 사진이 있으면 사진도 표시

### 매출 수정 / 삭제

- 상세 패널에서 **[수정]** 버튼을 누르면 수정 폼이 열립니다.
- **[삭제]** 버튼을 누르면 확인 팝업이 뜹니다. 실수로 삭제하는 것을 방지해요.

### 매출에 사진 첨부하기

매출을 등록한 직후, 사진을 첨부할 수 있는 안내가 나타납니다.

- 최대 10장까지 업로드 가능
- 3MB 초과 사진은 자동으로 압축됩니다.
- 업로드한 사진은 사진첩에서도 확인할 수 있어요.

### 카테고리/결제방식 설정

매출 페이지 상단의 **톱니바퀴 아이콘**을 클릭하면 설정 팝업이 열립니다.

- **카테고리 관리**: 추가, 수정, 삭제, 순서 변경, 색상 지정
- **결제방식 관리**: 추가, 수정, 삭제, 순서 변경, 색상 지정

여기서 설정한 카테고리와 결제방식이 매출 등록 폼의 드롭다운에 나타나요.

## 3. 지출 관리

꽃 매입비, 배달비, 임대료 등 모든 지출을 기록하고 관리합니다.

### 지출 등록하기

1. 지출 페이지 상단의 **[+ 지출 등록]** 버튼 클릭
2. 다음 항목을 입력합니다:

| 항목 | 설명 | 필수 |
|------|------|------|
| 날짜 | 지출 발생 날짜 | O |
| 카테고리 | 지출 종류 (꽃 매입, 배달, 광고, 임대료 등) | O |
| 항목명 | 구체적인 지출 내용 (예: 장미 100송이) | O |
| 단가 | 개당 가격 (콤마 자동 입력) | O |
| 수량 | 개수 (기본값: 1) | O |
| 합계 | 자동 계산 (단가 x 수량) | 자동 |
| 결제방식 | 카드, 현금, 계좌이체 등 | O |
| 카드사 | 카드 결제 시 카드사 선택 | 카드 결제 시 |
| 거래처 | 거래처명 (선택) | - |
| 메모 | 추가 메모 사항 | - |

3. **[등록]** 클릭

> **합계 자동 계산**: 단가와 수량을 입력하면 합계가 자동으로 계산됩니다.
> 예: 단가 3,000원 x 수량 100 = 합계 300,000원

### 지출 목록 조회

매출과 동일한 필터링 기능을 제공합니다:

- **월 필터**: 연/월 선택
- **결제방식 필터**: 결제방식별 조회
- **카테고리 필터**: 지출 종류별 조회
- **검색**: 항목명, 거래처, 메모로 검색

### 지출 요약 카드

- 전체 지출 합계
- 카테고리별 합계와 아이콘 표시

| 카테고리 아이콘 | 의미 |
|------|------|
| 🛒 | 꽃 매입 |
| 🚚 | 배달 |
| 📢 | 광고 |
| 🏠 | 임대료 |
| ⚡ | 공과금 |
| 📦 | 소모품 |
| 👛 | 기타 |

### 지출 상세 / 수정 / 삭제

- 목록에서 지출을 클릭하면 상세 패널이 열립니다.
- 단가 x 수량 = 합계 계산 내역을 확인할 수 있어요.
- 수정/삭제 모두 가능합니다.

### 카테고리/결제방식 설정

매출과 동일하게 톱니바퀴 아이콘으로 지출 전용 카테고리와 결제방식을 관리할 수 있습니다.

## 4. 고객 관리

고객 정보와 구매 이력을 체계적으로 관리합니다.

### 고객 등록하기

1. 고객 관리 페이지 상단의 **[+ 고객 등록]** 버튼 클릭
2. 다음 항목을 입력합니다:

| 항목 | 설명 | 필수 |
|------|------|------|
| 이름 | 고객 이름 | O |
| 전화번호 | 010-0000-0000 (자동 포맷, 중복 검사) | O |
| 등급 | 신규, 단골, VIP, 블랙 | O |
| 메모 | 고객 특이사항 (예: 알레르기, 선호 꽃 등) | - |

> **전화번호 중복 검사**: 입력 중에 이미 등록된 전화번호인지 실시간으로 확인합니다.
> 같은 전화번호로 두 명의 고객을 등록할 수 없어요.

> **자동 등록**: 매출 등록 시 고객명 + 전화번호를 입력하면 고객이 자동으로 등록됩니다.
> 이미 같은 전화번호가 있으면 기존 고객과 자동 연결돼요.

### 고객 등급 시스템

| 등급 | 배지 | 설명 |
|------|------|------|
| 신규 | 기본 | 처음 등록된 고객 |
| 단골 | 🌟 노란색 | 자주 방문하는 고객 |
| VIP | 👑 보라색 | 최우수 고객 |
| 블랙 | ⚠️ 빨간색 | 주의가 필요한 고객 |

등급은 수동으로 변경할 수 있습니다.

### 고객 목록 조회

- **등급 필터**: 전체, 신규, 단골, VIP, 블랙
- **검색**: 이름 또는 전화번호로 검색

### 고객 목록에 표시되는 정보

| 항목 | 설명 |
|------|------|
| 이름 | 등급 배지와 함께 표시 |
| 전화번호 | 010-1234-5678 형식 |
| 구매 횟수 | 총 구매 건수 |
| 구매 총액 | 총 구매 금액 |
| 최근 구매일 | 마지막 구매 날짜 |

### 고객 상세 보기

목록에서 고객을 클릭하면 상세 패널이 열립니다.

- 고객의 모든 정보를 확인할 수 있어요.
- **구매 이력 테이블**: 해당 고객의 모든 매출 내역 (날짜, 카테고리, 금액, 결제방식)
  - 각 매출을 클릭하면 매출 상세 페이지로 이동할 수 있어요.
- **[매출 등록]** 버튼: 클릭하면 매출 페이지로 이동하면서 고객 정보가 자동으로 채워집니다.

> **활용 팁**: 단골 고객이 전화로 주문하면, 고객 상세에서 이전 구매 이력을 확인하고
> 바로 매출 등록 버튼을 눌러서 빠르게 처리할 수 있어요.

## 5. 입금 대조

카드/네이버페이 결제 건의 실제 입금 여부를 확인하고 관리합니다.

### 입금 대조란?

카드로 결제하면 결제 금액에서 수수료를 뺀 금액이 며칠 뒤 통장에 입금됩니다.
입금 대조 페이지는 "어떤 매출이 입금됐고, 어떤 매출이 아직 안 됐는지"를 관리하는 곳이에요.

### 요약 카드

| 카드 | 내용 | 색상 |
|------|------|------|
| 카드 매출 합계 | 전체 카드/네이버페이 매출 | 기본 |
| 미확인 | 아직 입금 확인 안 된 건수 + 금액 | 주황 |
| 확인 완료 | 입금 확인된 건수 + 금액 | 초록 |
| 대조율 | 완료 비율 (프로그레스 바) | - |

### 미확인 탭

입금 확인이 안 된 매출 목록입니다.

| 표시 항목 | 설명 |
|------|------|
| 날짜 | 매출 발생 날짜 |
| 상품명 | 매출 상품명 (클릭하면 매출 상세로 이동) |
| 고객명 | 고객 이름 |
| 카드사 | 결제한 카드사 |
| 결제 금액 | 실제 결제 금액 |
| 예상 입금액 | 수수료를 뺀 금액 |
| 예상 입금일 | 영업일 기준 계산된 입금 예정일 |

**입금 확인하기**:
1. 통장에 입금된 건을 찾아서 **체크박스**를 선택합니다.
2. 여러 건을 한 번에 선택할 수 있어요.
3. 하단에 선택한 건수와 금액 합계가 표시됩니다.
4. **[입금 확인]** 버튼을 클릭하면 일괄 처리됩니다.

### 확인 완료 탭

이미 입금 확인된 매출 목록입니다.

- 입금 확인 날짜가 함께 표시됩니다.
- 실수로 확인한 경우 **[되돌리기]** 버튼으로 미확인 상태로 되돌릴 수 있어요.

### 필터링

- **월 필터**: 원하는 연/월의 매출만 보기
- **카드사 필터**: 특정 카드사만 보기
- **정렬**: 날짜, 금액, 카드사 기준으로 오름차순/내림차순

> **참고**: 현금/계좌이체 결제는 입금 대조 대상이 아닙니다.
> 카드와 네이버페이 결제만 이 페이지에 나타나요.

## 6. 예약 캘린더

고객 예약을 캘린더 형태로 관리합니다.

### 캘린더 화면

- 월간 달력 형태로 표시됩니다.
- 예약이 있는 날짜에는 상태별 색상 점이 표시돼요.
- 이전/다음 달 이동이 가능합니다.
- 날짜를 클릭하면 해당 날짜의 예약 목록이 나타납니다.

### 예약 상태

| 상태 | 색상 | 의미 |
|------|------|------|
| 대기 | 노란색 | 예약 접수됨, 확정 전 |
| 확정 | 파란색 | 고객이 확인함 |
| 완료 | 초록색 | 매출 전환 완료 |
| 취소 | 회색 | 예약 취소됨 |

### 예약 등록하기

1. 캘린더에서 원하는 날짜를 클릭
2. **[+ 예약 등록]** 버튼 클릭
3. 다음 항목을 입력합니다:

| 항목 | 설명 | 필수 |
|------|------|------|
| 날짜 | 예약 날짜 | O |
| 시간 | 예약 시간 (HH:MM) | O |
| 제목 | 예약 내용 (예: 장미 꽃다발) | O |
| 고객명 | 고객 이름 | O |
| 전화번호 | 고객 전화번호 | - |
| 설명 | 상세 내용 (예: 빨간 장미 30송이, 리본 포함) | - |
| 예상 금액 | 예상 결제 금액 | - |
| 상태 | 대기, 확정 중 선택 | O |

4. **[등록]** 클릭

### 예약을 매출로 전환하기

고객이 방문해서 결제를 완료하면, 예약을 바로 매출로 전환할 수 있습니다.

1. 캘린더에서 해당 예약의 **[매출 등록]** 버튼 클릭
2. 팝업에 예약 정보(날짜, 고객명, 금액)가 자동으로 채워져 있음
3. 카테고리, 결제방식만 선택
4. **[매출 등록]** 클릭

이렇게 하면 한 번에 세 가지가 처리됩니다:
- 매출이 등록됩니다.
- 예약 상태가 자동으로 **"완료"**로 바뀝니다.
- 신규 고객이면 고객도 자동으로 등록됩니다.

> 완료된 예약에는 **"매출 확인"** 링크가 생겨서,
> 나중에 이 예약이 어떤 매출로 연결됐는지 바로 확인할 수 있어요.

### 예약 수정 / 삭제

- 예약 목록에서 **[수정]** 버튼으로 정보를 변경할 수 있어요.
- **[삭제]** 버튼을 누르면 확인 팝업이 뜹니다.

## 7. 사진첩

꽃다발, 꽃바구니 등 작업 사진을 카드 형태로 관리합니다.

### 사진 카드란?

사진첩은 "사진 카드" 단위로 관리됩니다. 하나의 카드에 여러 장의 사진과 정보를 담을 수 있어요.

| 항목 | 설명 |
|------|------|
| 제목 | 카드 제목 (예: 빨간 장미 꽃다발) |
| 설명 | 상세 설명 |
| 사진 | 최대 10장 (jpg, png, webp, heic 지원) |
| 태그 | 여러 개의 태그 선택 가능 (색상별) |
| 매출 연결 | 특정 매출과 연결 가능 |

### 사진 업로드하기

1. 사진첩 페이지 상단의 **[+ 사진 등록]** 버튼 클릭
2. 드래그 앤 드롭 또는 클릭으로 사진 업로드
3. 제목, 설명, 태그를 입력
4. **[등록]** 클릭

> **자동 압축**: 3MB를 초과하는 사진은 자동으로 압축됩니다. 별도 작업이 필요 없어요.

> **매출에서 바로 업로드**: 매출 등록 직후 사진 첨부 안내가 나타납니다.
> 여기서 업로드하면 매출과 자동으로 연결된 사진 카드가 만들어져요.

### 사진 보기

- 반응형 그리드로 사진 카드가 나열됩니다.
- 카드를 클릭하면 전체 화면 뷰어가 열려요.
- 좌우 화살표로 사진을 넘길 수 있습니다.

### 태그로 필터링

- 상단의 태그 버튼을 클릭해서 특정 태그의 사진만 볼 수 있어요.
- 각 태그 옆에 해당 태그의 사진 카드 수가 표시됩니다.

### 태그 관리

사진첩 페이지의 **태그 관리** 버튼을 클릭하면:

- 새 태그를 만들 수 있어요 (이름 + 색상 선택).
- 9가지 프리셋 색상 중 선택: 흰색, 분홍, 빨강, 노랑, 보라, 남색, 청록, 주황, 회색
- 기존 태그의 이름/색상을 수정하거나 삭제할 수 있어요.

### 사진 카드 수정 / 삭제

- 카드 상세 화면에서 **[수정]**, **[삭제]** 버튼을 사용합니다.
- 삭제하면 저장소의 사진 파일도 함께 삭제됩니다.

## 8. 설정

시스템 전반의 설정을 관리합니다.

### 카드사 수수료 설정

카드사별 수수료율과 입금 소요일을 설정합니다.

| 카드사 | 설정 항목 |
|------|------|
| 신한, 국민, 삼성, 현대, 롯데, 하나, 우리, NH농협, BC | 수수료율 (%), 입금 소요일 (영업일) |

> 이 설정은 매출 등록 시 예상 입금액과 예상 입금일 자동 계산에 사용됩니다.
> 입금 대조 페이지에서도 이 설정값을 기준으로 표시해요.

**변경 방법**:
1. 설정 페이지에서 카드사 목록 확인
2. 수수료율(%)과 입금 소요일(영업일)을 수정
3. 변경 즉시 저장됩니다.

### 매출 카테고리/결제방식 설정

매출 페이지의 톱니바퀴 아이콘에서 관리합니다.

- **카테고리**: 상품 종류 (예: 꽃다발, 꽃바구니, 화환 등)
- **결제방식**: 결제 수단 (예: 카드, 현금, 계좌이체, 네이버페이 등)

각 항목마다:
- 이름(라벨)과 고유값(value) 설정
- 색상 지정 (배지 색상으로 사용)
- 드래그로 순서 변경 가능
- 추가, 수정, 삭제 가능

### 지출 카테고리/결제방식 설정

지출 페이지의 톱니바퀴 아이콘에서 관리합니다. 매출 설정과 동일한 방식이에요.

## 9. 기능 연결 가이드

헤이즐 어드민의 각 기능은 서로 연결되어 있어, 한 곳에서 시작해서 다른 곳으로 자연스럽게 이동할 수 있습니다.

### 9-1. 예약에서 바로 매출 등록하기

**상황**: 고객이 예약한 꽃다발을 픽업하고 결제했을 때

**이전 방식** (불편했던 점)
1. 캘린더에서 예약 확인
2. 매출 페이지로 이동
3. 날짜, 고객명, 전화번호, 금액을 다시 입력
4. 매출 등록
5. 다시 캘린더로 돌아가서 예약 상태를 "완료"로 변경

**지금은 이렇게 하세요**
1. 캘린더에서 해당 예약의 **[매출 등록]** 버튼 클릭
2. 팝업이 뜨면서 예약 정보(날짜, 고객, 금액)가 자동으로 채워져 있음
3. 상품 카테고리와 결제방식만 선택
4. **[매출 등록]** 클릭

끝. 매출이 등록되고, 예약 상태도 자동으로 "완료"로 바뀝니다.

### 9-2. 대시보드에서 바로 매출 등록하기

**상황**: 대시보드에서 오늘 예약 목록을 보다가, 이미 결제 완료된 예약이 있을 때

1. 대시보드 > 오늘 예약 섹션에서 **"매출 등록하기"** 링크 클릭
2. 자동으로 캘린더로 이동하면서 매출 등록 팝업이 열림
3. 카테고리, 결제방식 선택 후 등록

> 대기/확정 상태인 예약에만 "매출 등록하기" 링크가 표시됩니다.
> 이미 완료된 예약이나 취소된 예약에는 표시되지 않아요.

### 9-3. 고객 상세에서 매출 등록하기

**상황**: 단골 고객이 전화로 주문해서, 고객 정보를 먼저 확인한 뒤 매출을 등록하고 싶을 때

1. 고객 관리 > 해당 고객 클릭 > 상세 패널
2. **[매출 등록]** 버튼 클릭
3. 매출 페이지로 이동하면서 등록 폼이 자동으로 열림
4. 고객명, 전화번호가 이미 채워져 있음
5. 상품, 금액, 결제방식만 입력하면 끝

> 고객 상세 패널에서 구매 이력도 함께 볼 수 있어서,
> "지난번에 뭘 주문하셨죠?" 할 때 바로 확인 가능합니다.

### 9-4. 입금 대조에서 원본 매출 확인하기

**상황**: 카드 입금 대조를 하다가, 이 건이 어떤 매출인지 상세하게 보고 싶을 때

1. 입금 대조 페이지에서 상품명 옆의 **링크 아이콘** 클릭
2. 해당 매출의 상세 정보가 있는 매출 페이지로 바로 이동

### 9-5. 매출에서 사진첩으로 연결

**상황**: 매출로 등록한 꽃다발의 사진을 기록하고 싶을 때

1. 매출 등록 직후 사진 첨부 안내에서 사진 업로드
2. 사진첩에 자동으로 사진 카드가 생성되며 해당 매출과 연결됨
3. 나중에 사진첩에서 사진 카드를 열면 연결된 매출 정보를 볼 수 있어요.
4. 매출 상세에서도 연결된 사진을 확인할 수 있어요.

### 전체 흐름 한눈에 보기

```
예약 등록 (캘린더)
  |
  +-- 고객이 방문/결제 --> [매출 등록] 버튼 클릭
  |                         +-- 예약 정보 자동 채움
  |                         +-- 매출 생성
  |                         +-- 예약 상태 -> 완료 (자동)
  |                         +-- 고객 등록 (자동, 신규 고객인 경우)
  |                         +-- 사진 첨부 -> 사진첩에 카드 생성
  |
  +-- 카드 결제인 경우 --> 입금 대조 페이지에서 확인
                            +-- [매출 확인] 링크로 원본 매출 조회


고객 관리
  |
  +-- 고객 상세 보기 --> 구매 이력 확인
  |                       +-- [상세 보기] 링크로 매출 페이지 이동
  |
  +-- [매출 등록] 버튼 --> 매출 페이지 (고객 정보 자동 채움)


사진첩
  |
  +-- 사진 카드 상세 --> 연결된 매출 확인
  +-- 매출 상세에서 --> 연결된 사진 확인


대시보드
  |
  +-- 오늘 예약 --> "매출 등록하기" 클릭 --> 캘린더 매출 전환
  +-- 최근 매출 --> 매출 페이지로 이동
  +-- 입금 미확인 알림 --> 입금 대조 페이지로 이동
  +-- 월간 분석 --> 카테고리/결제/채널/고객/지출 통계
```

## 10. 내보내기

매출, 지출, 고객 데이터를 파일로 내보낼 수 있습니다.

### 지원 형식

| 형식 | 용도 |
|------|------|
| CSV | 엑셀이나 구글 시트에서 열기, 데이터 가공 |
| Excel (.xlsx) | 엑셀에서 바로 열기, 서식 포함 |
| PDF | 인쇄용, 보관용 |

### 내보내기 방법

1. 매출/지출/고객 페이지에서 **[내보내기]** 버튼 클릭
2. 원하는 형식 (CSV / Excel / PDF) 선택
3. 파일이 자동으로 다운로드됩니다.

> 현재 적용된 필터(월, 카테고리, 결제방식 등)가 반영된 데이터가 내보내집니다.
> 예: 2025년 1월 카드 매출만 필터링한 상태에서 내보내면, 해당 데이터만 파일에 포함돼요.

### 내보내기 항목

**매출**: 날짜, 카테고리, 금액, 결제방식, 유입 채널, 고객명, 메모

**지출**: 날짜, 카테고리, 금액, 결제방식, 수량, 항목명, 거래처, 메모

**고객**: 이름, 전화번호, 등급, 구매 횟수, 구매 총액, 최근 구매일, 메모

## 12. 자주 묻는 질문

### 매출 관련

**Q. 예약 없이 바로 매출을 등록해도 되나요?**
네. 기존처럼 매출 페이지에서 직접 등록하는 것도 그대로 가능합니다.
예약 연결은 선택 사항이에요.

**Q. 매출 등록할 때 고객이 자동으로 생성되나요?**
네. 전화번호가 있는 고객은 자동으로 생성됩니다.
이미 등록된 전화번호라면 기존 고객과 자동으로 연결돼요.

**Q. 매출을 삭제하면 연결된 고객도 삭제되나요?**
아니요. 매출을 삭제해도 고객 정보는 그대로 유지됩니다.
고객의 구매 횟수와 총액만 자동으로 업데이트돼요.

**Q. 카드 수수료는 어떻게 바꾸나요?**
설정 페이지에서 카드사별 수수료율(%)과 입금 소요일을 변경할 수 있어요.
변경하면 이후 등록하는 매출부터 적용됩니다.

### 예약 관련

**Q. 예약을 매출로 전환한 뒤, 예약을 다시 수정할 수 있나요?**
네. 예약 정보 자체는 수정 가능합니다.
다만 연결된 매출은 매출 페이지에서 따로 수정해야 해요.

**Q. 실수로 매출을 등록했는데 예약 상태가 완료로 바뀌었어요.**
매출을 삭제하면 됩니다. 예약 상태는 캘린더에서 직접 다시 변경해주세요.

**Q. 예약에 고객 전화번호를 꼭 입력해야 하나요?**
아니요. 전화번호는 선택 사항입니다. 다만 입력하면 매출 전환 시 고객 자동 등록에 활용돼요.

### 고객 관련

**Q. 고객 등급은 자동으로 바뀌나요?**
아니요. 등급은 수동으로 변경해야 합니다.
고객 상세에서 등급을 직접 선택해주세요.

**Q. 같은 전화번호로 두 명의 고객을 등록할 수 있나요?**
아니요. 전화번호는 고객 식별의 기준이라서 중복 등록이 불가능해요.
등록 시 실시간으로 중복 검사가 이루어집니다.

### 입금 대조 관련

**Q. 현금 매출도 입금 대조에 나오나요?**
아니요. 카드와 네이버페이 결제만 입금 대조 대상입니다.
현금, 계좌이체는 별도의 입금 대조가 필요 없어요.

**Q. 입금 확인을 잘못 눌렀어요.**
확인 완료 탭에서 해당 건의 **[되돌리기]** 버튼을 누르면
미확인 상태로 되돌릴 수 있어요.

### 사진첩 관련

**Q. 사진 용량 제한이 있나요?**
한 장당 3MB까지 업로드할 수 있어요. 3MB를 초과하는 사진은 자동으로 압축됩니다.
한 카드당 최대 10장까지 첨부 가능합니다.

**Q. 지원하는 사진 형식은?**
JPG, JPEG, PNG, WebP, HEIC, HEIF 형식을 지원합니다.
아이폰으로 촬영한 HEIC 사진도 바로 업로드할 수 있어요.

### 내보내기 관련

**Q. 내보내기하면 전체 데이터가 나오나요?**
현재 적용된 필터가 반영됩니다. 전체 데이터를 내보내려면 필터를 "전체"로 설정한 뒤 내보내기하세요.

**Q. 내보낸 파일은 어디에 저장되나요?**
브라우저의 기본 다운로드 폴더에 저장됩니다.

### 기타

**Q. 모바일에서도 사용할 수 있나요?**
네. 모든 페이지가 모바일 화면에 맞게 반응형으로 표시됩니다.
사이드바는 접힌 상태로 시작하고, 필요할 때 열 수 있어요.

**Q. 다크모드 설정은 저장되나요?**
네. 브라우저에 저장되어 다음 접속 시에도 유지됩니다.

**Q. 로그아웃은 어떻게 하나요?**
사이드바 하단 또는 헤더의 프로필 영역에서 로그아웃할 수 있어요.
